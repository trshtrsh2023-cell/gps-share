# 📍 مشارك المواقع - GPS Image Sharer

تطبيق ويب لمشاركة الصور مع مواقعها الجغرافية

## 🚀 المميزات

- ✅ رفع صور متعددة
- ✅ التقاط صور مباشرة من الكاميرا
- ✅ استخراج GPS تلقائي من بيانات EXIF
- ✅ عرض الخرائط التفاعلية
- ✅ إنشاء صورة مركبة (الصورة + الخريطة + الموقع)
- ✅ مشاركة مباشرة للواتساب/سناب/تليجرام
- ✅ واجهة عربية RTL كاملة
- ✅ متوافق مع الموبايل والآيفون

## 📦 التثبيت

```bash
npm install
```

## 🏃 التشغيل

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000)

## 🌐 النشر على Vercel

1. ادفع المشروع إلى GitHub
2. اذهب إلى [vercel.com](https://vercel.com)
3. استورد المشروع من GitHub
4. اضغط Deploy

## 📱 الصلاحيات المطلوبة

- **الموقع (Location)**: لحفظ GPS في الصور الملتقطة
- **الكاميرا (Camera)**: لالتقاط صور جديدة

## 🛠️ التقنيات المستخدمة

- Next.js 14
- React 18
- TypeScript
- Leaflet (خرائط)
- EXIF.js (استخراج GPS)
- html2canvas (إنشاء صور مركبة)

## 📝 ملاحظات

- يعمل على HTTPS فقط (للصلاحيات)
- localStorage للتخزين المحلي
- يمكن تحويله لاستخدام Supabase لاحقًا
